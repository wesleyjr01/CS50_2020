# Regular Expressions
* &nbsp;&nbsp;**.** &nbsp;&nbsp; any character
* &nbsp;&nbsp;**.\*** &nbsp;&nbsp;0 or more characters
* &nbsp;&nbsp;**?** &nbsp;&nbsp; optional
* &nbsp;&nbsp;**^** &nbsp;&nbsp; start of input
* &nbsp;&nbsp;**$** &nbsp;&nbsp; end of input
